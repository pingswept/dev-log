Start at https://www.home-assistant.io

Burn the image https://github.com/home-assistant/operating-system/releases/download/7.0/haos_rpi4-64-7.0.img.xz to an SD card using Balena Etcher.

Boot a Raspberry Pi off the SD card, connect via Ethernet, and visit http://homeassistant.local:8123/

Follow some setup instructions to set a username and password.

Install the Mosquitto MQTT Broker as a Supervisor add-on by navigating to: `Configuration > Settings > Info > scroll down to Home Assistant Supervisor > Manage > Add-on Store`

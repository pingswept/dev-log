Building Sean McIntyre's DMX library

    git clone https://github.com/rascalmicro/beaglebone-DMX
    cd pasm/pasm_source
    ./linuxbuild # builds executable called pasm at beaglebone-DMX/pasm/pasm

Can successfully execute

    ./pasm
